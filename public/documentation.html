Documentation - Movies

REQUEST URL HTTP METHOD EXAMPLE REQUEST RESPONSE
Return a list of ALL movies to the user /movies GET /movies A JSON object containing data of all movies
Return data about a single movie by title to the user /movies/[title] GET /movies/The Dark Knight A JSON object containing data of a single movie
Return data about a genre (e.g.,“Thriller”) /movies/genre/[genre] GET /movies/genre/Thriller A JSON object describing the selected genre
Return data about a director (bio, birth year, death year) by name /movies/directors/[director] GET /movies/directors/Christopher Nolan A JSON object containing data of selected director
